Short Description about my Webpage:

"Iron fist" is a modern gym web page designed using HTML(index.html) and CSS(style.css).

The deep yellow (#FCB042), white and grey [rgba(4,9,30,0.7)] color co-ordination in the animations and the button transitions adds a vibrant and professional feel to the web page.
It features 7 important sections required for a gym and a smooth user experience.

The Home page introduces the gym and provides a list of locations the gym is situated at.
The Features highlights its top amenities.
Users can explore various workout plans and understand about them using the a customized and an embedded YouTube video provided.
There are 2 user Reviews added with their audios created using text-to-speech.
The Contact us and Mail us sections provide a way to call and send an E-mail.
The footer includes more information of the gym as the About us section.

The web page is designed with responsive styling and interactive transitions which ensures dynamic user experience.
It is full optimized for smaller screens like mobile phones and tablets with a responsive menu making it accessible on all the devices.
